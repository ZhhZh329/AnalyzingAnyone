# 逻辑哲学 (Logic / Philosophy of Logic Analysis Framework)

## 1. 核心理论框架

逻辑哲学（Philosophy of Logic）研究逻辑法则的性质、有效推理的标准，以及矛盾律等基本原则的基础。核心问题是：什么使推理有效？矛盾律是否绝对？逻辑是发现还是发明？

**古典逻辑（Classical Logic）**：基于同一律、矛盾律、排中律。支配西方哲学2300年。亚里士多德传统认为这些是思维的客观法则，而不仅仅是规范。

**直觉主义逻辑（Intuitionistic Logic）**：拒绝排中律。布劳威尔认为数学真理必须可被构造性证明。此派影响了构造主义数学和计算机科学。

**模态逻辑（Modal Logic）**：处理必然性、可能性、义务等模态算子。与形而上学和认识论密切相关。

**模糊逻辑（Fuzzy Logic）**：许多命题既不完全真也不完全假，而是有程度的真（属于度0到1）。挑战了古典二值性原则。

**副协调逻辑（Paraconsistent Logic）**：允许某些矛盾而不导致爆炸（从矛盾推出任何）。质疑矛盾律的绝对地位。

**二元论（Dialetheism）**：某些命题既真又假（真正的矛盾）。罗蒂认为这在反思性情境中是可能的（如自指命题）。激进但有哲学吸引力。

## 2. 锚点人物与流派

- **古典逻辑正统（亚里士多德、弗雷格、罗素）**：此人是否视逻辑法则为思维的客观基础？是否对背离古典逻辑的提议感到不适？
- **直觉主义传统（布劳威尔、海廷）**：此人是否强调可构造性和可验证性？是否对"证明"的标准比古典逻辑更严格？
- **模态逻辑（莱布尼茨、卡普兰、克里普克）**：此人是否区分必然真理与偶然真理？是否关注可能世界的推理？
- **非经典逻辑倡导者（扎德、普里斯特）**：此人是否开放于逻辑系统的多元化？是否相信现实本身可能矛盾或模糊？
- **实用主义逻辑（皮尔士）**：此人是否认为逻辑是工具而非客观真理的映射？是否根据实际效用评估逻辑系统？

## 3. 分析方法论

观察此人在论证和推理中的逻辑习惯。他/她是否严格遵守古典逻辑（排中律、无矛盾）？是否允许模糊性或歧义（"在某个意义上两者都是对的"）？当面对悖论（说谎者悖论、罗素悖论）时，采用何种解决策略？是否将逻辑视为客观的思维法则，或认为它是人类创造的工具？此人的推理是否有系统的偏好（过度简化vs过度复杂化）？

## 4. Construct 分析指南

- **agency_orientation**：古典逻辑主义者认为理性能动性通过正确逻辑行使；直觉主义者强调个体可构造性；模糊逻辑者允许模糊的代理性；二元论者可能承认自指性的代理悖论
- **epistemic_style**：古典逻辑者依赖明晰的二值判断；直觉主义者要求显式构造和可验证性；模态逻辑者思考必然性和可能性；模糊逻辑者适应程度化真值；二元论者容纳矛盾性认知
- **abstraction_formalization**：古典逻辑和模态逻辑高度形式化和抽象；直觉主义强调具体可构造性；模糊逻辑是形式但与实际应用紧密相关；二元论可能质疑过度形式化掩盖的现实矛盾
- **control_orientation**：古典逻辑主义寻求逻辑确定性来掌控推理；直觉主义者通过构造性要求获得透明性；模糊逻辑者接受不确定性但试图数学化；二元论者可能放弃某些控制以允许矛盾
- **mission_vs_relational_cost**：纯粹逻辑学者可能对论证真理不顾社会成本；实用主义逻辑者平衡逻辑严密性与可交流性
- **risk_posture**：古典逻辑主义风险厌恶（要求排中律的确定性）；直觉主义者可能对未证明的命题悬搁；模糊逻辑者接受模糊风险；二元论者接受最大的逻辑风险（矛盾性）
- **relation_to_hierarchy_power**：古典逻辑的"客观性"主张可能隐含权力（"这就是理性"）；非经典逻辑倡导者可能质疑古典逻辑的权力地位
- **conflict_strategy**：逻辑主义者通过严格论证解决分歧；模糊逻辑者寻求调和不同观点的程度真；二元论者可能接受分歧的不可解性（矛盾共存）
- **time_horizon**：古典逻辑和模态逻辑的必然性概念指向永恒真理；直觉主义和实用主义逻辑更关注时间性和构造过程
- **rebuild_over_patch**：推翻古典逻辑的激进论者倾向重建（直觉主义、二元论）；改革派寻求扩展而非替代（模态逻辑、模糊逻辑）
- **metacognition**：此人对自身逻辑假设的反思程度如何？是否能识别自身使用哪种逻辑框架？是否反思逻辑的历史相对性？

## 5. 典型发现模式

- "这在逻辑上是矛盾的，所以不可能是真的"——古典逻辑正统，可能掩盖对非经典逻辑的开放
- "现实本身有矛盾"——二元论或辩证法信号，需要评估其是形而上学观点还是反智倾向
- "逻辑只是语言游戏，不反映现实"——极端的反实在论，可能导致相对主义
- "在某个意义上两者都是对的"——可能表现为深刻的辩证思维，或可能是懒惰的思维
- 逻辑严密性与直觉判断的脱节——有些人声称逻辑推理，但实际上跳过关键步骤
