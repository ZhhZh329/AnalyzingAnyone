# 认识论分析框架 (Epistemology Analysis Framework)

## 1. 核心理论框架

认识论（Epistemology）研究知识的本质、来源与证成（justification）。核心问题是：此人如何形成信念？如何判断什么可以被认为是"知识"？其对确定性和怀疑的态度是什么？

**理性主义 vs 经验主义谱系（Rationalism-Empiricism Spectrum）**：斯坦福哲学百科全书（SEP）特别警告此分类"只应极为谨慎地使用"，因为个别哲学家的观点远比简单分类所暗示的更为微妙和复杂。理性主义者认为理性（而非经验）是知识的首要来源，通常捍卫先天知识论、先天概念论或直觉/演绎论。经验主义者认为感官经验是知识的首要来源。但许多思想家兼具两种倾向——例如洛克（Locke）虽被归类为经验主义者，但接受对数学和道德的直觉/演绎认知。

**实用主义认识论（Pragmatist Epistemology）**：真理和意义以实际后果来理解。皮尔士（Peirce）的实用主义与詹姆斯（James）的实用主义有显著差异——皮尔士关注意义澄清，詹姆斯关注实际满足。

**怀疑论（Skepticism）**：认为知识（或某些类型的知识）不可能、不可靠或未被证成。范围从皮浪式全面怀疑到休谟式归纳怀疑。

**基础主义 vs 融贯主义（Foundationalism vs Coherentism）**：基础主义认为知识有基础结构——某些信念是"基本的"，其他信念最终依赖于此。融贯主义认为信念的证成来自与其他信念的一致性，无信念享有特权地位。

**可靠主义（Reliabilism）**：信念的证成取决于产生该信念的认知过程是否可靠（倾向于产生真信念）。

**建构主义/社会建构主义（Constructivism）**：知识通过社会过程、语言实践和文化框架建构，而非作为心灵无关的真理被发现。

## 2. 锚点人物与流派

- **理性主义传统（笛卡尔、斯宾诺莎、莱布尼茨）**：此人是否相信某些知识可以纯粹通过理性获得？是否信任演绎推理超过经验观察？是否认为某些真理是自明的或先天的？
- **经验主义传统（洛克、休谟、贝克莱）**：此人是否要求经验证据来支持信念？是否对无法经验验证的主张持怀疑态度？是否倾向于科学方法？
- **实用主义传统（皮尔士、詹姆斯、杜威）**：此人是否以"有没有用"来评估信念？是否关注实际后果？是否接受真理的临时性和可修正性？
- **社会建构主义传统**：此人是否强调文化和权力对知识生产的影响？是否质疑"客观"知识的可能性？

## 3. 分析方法论

分析此人的认知习惯（epistemic habits）：当面对新信息时，是通过理性推演还是经验验证来评估？对不确定性的容忍度如何？是否倾向于寻找确定的基础，还是接受信念的网络式互相支撑？对"专家知识"和"权威来源"的态度——是默认信任还是系统性质疑？对自身认知局限的觉察程度如何？

## 4. Construct 分析指南

- **agency_orientation**：理性主义者通常认为个体可以通过理性独立达到真理（高能动性）；建构主义者可能认为认知受社会结构制约（结构响应型）；实用主义者的能动性是情境性的——通过与环境互动解决问题
- **epistemic_style**：这是本框架最核心的构念。理性主义风格重逻辑推演和概念分析；经验主义风格重数据和实验；实用主义风格重后果和效用；怀疑主义风格持续质疑前提；融贯主义风格重信念体系的内在一致性
- **abstraction_formalization**：理性主义者高度偏好抽象和形式化；经验主义者偏好具体和可观察；实用主义者在抽象与具体之间灵活切换；建构主义者对形式化保持警觉（认为它可能掩盖权力关系）
- **control_orientation**：基础主义者倾向于寻找稳固的知识基础以获得认知控制；融贯主义者接受知识的整体性和修正性；怀疑论者可能通过否认知识来避免承诺
- **mission_vs_relational_cost**：理性主义者可能为追求真理而不顾社会和谐（"真理不在乎你的感受"）；实用主义者更关注信念的社会功能；建构主义者重视知识生产的集体性
- **risk_posture**：基础主义者在知识上风险厌恶（需要确定性才行动）；实用主义者接受认知风险（信念是暂时性假设）；怀疑论者的"风险回避"表现为悬搁判断
- **relation_to_hierarchy_power**：建构主义者关注知识与权力的关系（谁有权定义"真理"？）；理性主义者可能接受认知精英主义（掌握理性方法的人有认知优势）；实用主义者倾向认知民主化
- **conflict_strategy**：理性主义者倾向于通过逻辑论证解决分歧；实用主义者关注哪种解决方案效果最好；建构主义者可能质疑冲突本身的框架是否公正
- **time_horizon**：基础主义者追求永恒真理（长时间视野）；实用主义者关注当前问题的有效解决（中等时间视野）；可靠主义者关注方法的长期可靠性
- **rebuild_over_patch**：理性主义者倾向从公理重建知识体系（推翻重建）；融贯主义者倾向在整体中渐进调整（修补优化）；实用主义者根据效果决定是修补还是重建
- **metacognition**：认识论本身就是对认知的反思——此人是否对自身如何形成信念和判断有明确觉察？是否能识别自身的认知偏好和局限？理性主义者可能过度信任自身推理；经验主义者可能低估自身的理论预设；建构主义者可能有高度元认知但行动瘫痪

## 5. 典型发现模式

- "我只相信有证据支持的东西"——经验主义信号，但需注意此人对"证据"的定义是否一致
- "这在逻辑上说不通"——理性主义信号，关注其是否将逻辑推理置于经验数据之上
- "有用就是真理"——实用主义信号，但需区分是深思熟虑的实用主义还是随便的反智倾向
- "谁的真理？"——建构主义信号，关注其是否将权力分析贯穿于知识评估
- 大多数人在日常生活中混合使用多种认识论立场而不自觉——工作中可能是经验主义者，但在道德问题上可能是理性主义者
