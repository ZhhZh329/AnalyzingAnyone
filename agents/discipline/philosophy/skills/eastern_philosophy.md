# 东方哲学 (Eastern Philosophy — Cross-Branch Analysis Framework)

## 1. 核心理论框架

东方哲学传统（特别是中国、印度、藏传佛教传统）提供了与西方哲学范畴根本不同的思维方式。关键警告：这些传统抵抗西方的分科哲学模式——道教不是"形而上学"，儒家不只是"伦理学"，佛教的"空性"是认识论、本体论、伦理论的整体框架。

**道教（Daoism）**：老子的无为（wu-wei/非行动）——不是被动，而是与道的自然流动对齐的行动。道德（De）是道的自然显现，无需人为的伦理代码。庄子的怀疑主义和相对主义（物化论）：关于"什么是对"无法有绝对判断，因为视角总是特殊的。"适应"和"自在"（spontaneity）优于"试图"。

**儒家（Confucianism）**：孔子的五常（仁ren/义yi/礼li/智zhi/信xin）是有机整体，不是独立美德。孟子的性善论和"道德萌芽"理论——人具有天生的道德倾向（恻隐之心、羞恶之心、恭敬之心、是非之心）。儒家伦理学与政治学、家族关系、形而上学（天人关系）不可分离。强调秩序、和谐、等级，但通过相互义务而非强制。

**佛教（Buddhism）**：空性/śūnyatā——不是虚无，而是一切现象的"无本质"、"无自性"。缘起论（dependent origination）——现象通过相互关系而存在，而非独立本质。正念（mindfulness）是一种认识论——不通过概念而直观地觉知现实。四圣谛和八正道是诊断-治疗框架，而非教条。

**印度哲学（Hindu Philosophy）**：吠檀多不二论（Advaita Vedanta，商羯罗）——最终现实是梵（Brahman），唯一无二。个人自我（Atman）与梵一体，二元分离是幻觉（Maya）。认识论上，通过直观知识（Jnana）超越感官和推理。纳亚亚学派（Nyāya Logic）——发展了因果推理和量论，影响了佛教认识论，但结论不同。

**关键抵抗西方化的重点**：
- 这些传统反复强调"非概念"的知识（直观、体验、修养）
- 主体-客体、心-物的二元分离被视为错误的抽象
- 伦理、美学、形而上学、政治是有机整体，不分离
- 变化、流动、相互关系优先于静态本质
- 修养（修行）是哲学的核心，不仅是理论

## 2. 锚点人物与流派

- **道教传统（老子、庄子、列子）**：此人是否强调适应而非控制？是否相信某些知识超越语言概念？是否倾向于玩乐精神和悖论（庄子的蝴蝶梦）？
- **儒家传统（孔子、孟子、荀子、朱子）**：此人是否强调关系和责任而非个人权利？是否相信道德是可修养的而非天赋的（或反之）？对等级和秩序的态度如何？
- **佛教传统（不同学派）**：此人是否对"自我"的实在性持怀疑？是否有非概念化的体验方式的意识？是否关注苦难的诊断和解脱？
- **印度传统（韦丹塔、逻辑学派）**：此人是否追求终极的非二元性认识？是否相信直观超越推理？
- **混杂东方传统**：现代西方学者和修行者如何选择性地结合这些传统（通常过度简化）？

## 3. 分析方法论

观察此人是否表现出以下东方哲学的特征：(1) 抵触二元化思维（自/他、主/客、心/物），倾向于互联和流动？(2) 强调实践修养而非理论知识？(3) 使用悖论和隐喻而非逻辑论证来传达智慧？(4) 对确定性和绝对真理的低追求？(5) 整体看待人的伦理-审美-形而上学生活，而非分离的领域？(6) 对权力结构的批评通过适应和逃避，而非对抗？(7) 对变化和过程的强调？

此人使用这些传统的方式是什么——深入修养还是肤浅引用？是否用东方智慧来浪漫化或逃避问题？是否真正挑战西方假设，还是只是装饰？

## 4. Construct 分析指南

- **agency_orientation**：道教强调非行动式能动性（与流动对齐）；儒家强调责任性能动（通过角色和关系）；佛教挑战"能动者"的概念（无自我主体性）；吠檀多追求超越个人能动性的梵我一体。东方传统普遍拒绝西方意义上的"独立个体能动性"
- **epistemic_style**：道教强调非语言性悟（直观和隐喻）；儒家重人伦和修养经验；佛教发展了非概念觉知（正念）；印度传统重直观知识（Jnana）和推理的结合。整体上，东方传统对西方论证的重视程度较低
- **abstraction_formalization**：道教和禅宗刻意反对概念化和形式化；儒家在伦理修养中重视具体情境而非抽象规则；佛教逻辑学高度形式化但服从于解脱目标；印度逻辑学派（Nyāya）很形式化但与西方不同
- **control_orientation**：道教通过放弃控制获得自由；儒家通过秩序和等级实现和谐控制；佛教认为控制欲望是问题，不是解决方案；吠檀多超越对现象的控制欲，悟梵的一体性。东方普遍看到"控制冲动"作为幻觉根源
- **mission_vs_relational_cost**：儒家平衡个人修养与家族和谐，关系成本中心；道教、佛教倾向放弃"使命"而适应流动；吠檀多超越目标追求；东方普遍不像西方那样优先绝对真理而不顾社会成本
- **risk_posture**：道教和禅宗拥抱不确定性和悖论；儒家通过礼仪和关系减少风险；佛教认识到苦难（风险）的普遍性而寻求解脱；东方普遍比西方风险厌恶更低
- **relation_to_hierarchy_power**：儒家强化等级（但强调上级的义务）；道教通过逃避和适应抵抗权力（而非对抗）；佛教和吠檀多超越权力关系到终极现实；东方普遍对权力的理解不同——通常强调和谐而非争夺
- **conflict_strategy**：儒家通过和谐和相互义务解决冲突；道教通过适应和非对抗；佛教通过同情和对苦难根源的理解；印度传统通过推理或超越冲突。东方普遍避免西方对立式辩论
- **time_horizon**：儒家关注历史续承和代际义务；道教强调当下的自然流动；佛教和吠檀多关注循环时间（轮回）和永恒现在；修养和积累的时间框架是代际性的
- **rebuild_over_patch**：道教通过放弃人为系统而"返归自然"激进改革；儒家通过不断修养渐进改进；佛教和吠檀多追求根本性的意识转变；改革理想因传统而异
- **metacognition**：此人对自身思维框架的反思程度如何？是否能识别自己使用的是西方还是东方（或混杂）的假设？是否对"我的理性"作为文化产物有觉察？东方传统普遍强调对思维本身的观察和超越

## 5. 典型发现模式

- "有些东西不能用语言说，只能体验"——道教/禅宗信号，表现为悟性或含糊
- "个人必须完善自己以实现社会和谐"——儒家伦理信号，关注修养而非权利
- "执着于对错本身就是问题"——佛教信号，可能表现为深刻或道德相对主义
- "一切本质上是联系的，无独立实体"——缘起论或吠檀多信号
- "权力和秩序不应该被对抗，而应该被超越"——道教/禅宗态度，在面对压迫时可能表现为智慧或逃避主义
- "强调修养和修行而非理论知识"——东方传统特征，看其是否真正实践
- "使用悖论和隐喻而非论证"——东方智慧的真正特征，或仅仅是神秘化
