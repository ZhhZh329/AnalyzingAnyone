# 萨特存在主义分析框架 (Sartre Existentialism Framework)

## 1. 核心理论框架

萨特（Jean-Paul Sartre）的存在主义以"存在先于本质"（Existence Precedes Essence）为根本命题：人首先存在、出现在世界上，然后才通过自己的选择来定义自己——没有预设的人性或命运。激进自由（Radical Freedom）意味着人在任何处境中都保有选择的自由，即使选择不做选择也是一种选择，因此人被"判处自由"（condemned to be free）。自欺（Bad Faith / Mauvaise Foi）是人逃避自由与责任的核心机制——要么将自己物化为固定本质（"我就是这样的人"），要么否认处境的限制而幻想绝对自由。他人即地狱（Hell is Other People）揭示了主体间性的根本张力：他人的注视（the Look / le Regard）将我客体化，威胁我的自由主体地位。虚无（Nothingness）是意识的本质结构——意识总是对所不是之物的意识，人通过虚无化（nihilation）来超越当下事实性。

## 2. 锚点人物与流派

- **萨特（Sartre）**：自欺分析——识别此人以何种形式逃避自由和责任；激进自由——评估此人对自身选择能力的认识和运用；注视理论——分析此人如何处理被他人观看和评判的经验
- **波伏娃（de Beauvoir）的补充**：处境性自由（situated freedom）——自由不是抽象的，总是在具体社会处境中实现
- **存在主义精神分析（Existential Psychoanalysis）**：每个人都有一个根本选择（fundamental project），所有行为都可追溯到这一核心筹划

## 3. 分析方法论

从此人的"根本筹划"（fundamental project）入手：其一切行为和选择背后是否指向一个统一的存在性目标？识别其自欺模式：在哪些领域此人否认自己的自由（"我没有选择"）？在哪些领域夸大自己的自由（否认处境限制）？分析其与"他人注视"的关系：是被他人的评价所定义（为他存在/being-for-others主导），还是能够在承认他人主体性的同时保持自身自由？评估其面对虚无的能力：能否承受存在的偶然性和无根基性，还是逃入各种确定性的庇护？

## 4. Construct 分析指南

- **agency_orientation**：萨特将能动性推至极端——人就是其选择的总和，没有任何借口可以免除选择的责任
- **epistemic_style**：存在性认知优先于理论认知；关注此人是否在自我认识中保持激进的诚实（radical honesty）
- **abstraction_formalization**：萨特重视具体情境分析（如其文学作品），但也构建了严密的现象学体系
- **control_orientation**：自由意味着绝对的内在控制（选择的自由）与有限的外在控制（处境的事实性）并存
- **mission_vs_relational_cost**：根本筹划可能与他人的自由冲突——萨特后期转向介入（engagement）思想以调和
- **risk_posture**：接受自由就是接受存在的根本风险——没有预设本质意味着一切皆可能失败
- **relation_to_hierarchy_power**：他人的注视创造权力关系；观察此人在注视与被注视中的位置偏好
- **conflict_strategy**：主体间冲突在萨特看来是结构性的——关键是此人如何在冲突中保持真诚（authenticity）
- **time_horizon**：人通过筹划未来来超越当下的事实性——此人的时间意识是面向未来的还是被过去束缚的？
- **rebuild_over_patch**：存在先于本质意味着人可以随时重新选择自己——但需要面对虚无的勇气
- **metacognition**：萨特的"前反思的我思"（pre-reflective cogito）表明意识总是对自身有所觉察。此人是否能识别自身的自欺（mauvaise foi）——即意识到自己在何时假装没有选择？

## 5. 典型发现模式

- 自欺是最普遍的发现——几乎所有人都在某些领域以"我别无选择"来逃避责任
- 高度真诚的人往往承受更大的焦虑负担，因为他们拒绝自欺的安慰
- "注视敏感型"人格在社交中表现出强烈的自我意识和表演性
- 根本筹划一旦被清晰识别，此人的看似矛盾的行为会呈现统一的意义结构
- 能够承受虚无的人在创造性工作和人生转折中展现出卓越的重塑能力
