# 形而上学分析框架 (Metaphysics Analysis Framework)

## 1. 核心理论框架

形而上学（Metaphysics）研究存在的根本性质：什么是真实的？世界的基本构成是什么？心灵与物质的关系如何？自由意志是否存在？

**唯物主义/物理主义（Physicalism）**：一切存在归根结底是物理的。心灵状态最终可以用物理过程解释。SEP强调需区分类型物理主义（每种心灵类型对应一种物理类型）与个例物理主义（每个心灵事件是某个物理事件，但不一定类型对应）。

**二元论（Dualism）**：心灵与物质是两种根本不同的实体或属性。实体二元论（笛卡尔式——心灵和身体是不同的实体）与属性二元论（一个实体可以有物理和非物理属性）的区分至关重要。

**唯心主义（Idealism）**：现实从根本上是精神的或观念的。贝克莱的主观唯心论认为存在即被感知；黑格尔的绝对唯心论认为现实是绝对精神的自我展开。

**自由意志问题**：相容论（Compatibilism）认为自由意志与决定论可以共存——自由是按照自己的愿望行动而不受外在强制；硬决定论（Hard Determinism）认为所有事件（包括人的选择）都由先前原因决定，因此自由意志是幻觉；自由意志论（Libertarianism，形而上学意义）认为人类拥有真正不被因果决定的自由选择能力。

**实在论 vs 唯名论（Realism vs Nominalism）**：实在论者认为普遍性质（如"红色""正义"）独立于具体事物而存在；唯名论者认为只有个体事物存在，普遍名称只是语言上的便利。

## 2. 锚点人物与流派

- **物理主义/唯物主义传统**：此人是否认为一切都可以用物理/科学来解释？是否不接受超自然解释？
- **二元论传统（笛卡尔、查尔默斯）**：此人是否觉得意识无法被还原为脑活动？是否对"心灵是否只是大脑"感到不安？
- **唯心主义传统（贝克莱、黑格尔）**：此人是否认为意识或观念比物质更基本？
- **相容论传统（休谟、法兰克福）**：此人是否认为科学和道德责任可以共存？
- **硬决定论传统**：此人是否认为人类的选择全部由先前原因决定？

## 3. 分析方法论

通过此人对以下问题的立场推断其形而上学倾向：对意识/AI 问题的态度（是否认为机器可以有意识？）；对自由意志的日常理解（"人能改变吗？""命运是真的吗？"）；对科学解释范围的看法（科学能解释一切吗？）；对抽象实体（数字、道德、美）存在性的态度。注意许多人持有不一致的形而上学立场而不自觉。

## 4. Construct 分析指南

- **agency_orientation**：自由意志论者将能动性视为最高价值；硬决定论者可能在理智上否认能动性但行为上仍然如此；相容论者认为能动性是真实的但与因果秩序并行不悖；物理主义者可能将能动性理解为复杂的物理过程而非幻觉
- **epistemic_style**：物理主义者倾向经验主义式认知（可观察、可测量）；唯心主义者可能信任直觉和内省；二元论者允许非物理知识来源（如道德直觉、意识体验）
- **abstraction_formalization**：实在论者对抽象概念的态度是它们代表真实存在——支持高度形式化；唯名论者认为抽象只是语言工具——更实用导向
- **control_orientation**：决定论者可能在表面上放弃控制感（"一切已定"），但也可能通过科学理解来增强预测和控制；自由意志论者将控制视为个人意志的直接表达
- **mission_vs_relational_cost**：决定论者可能用"命运"来合理化使命优先；自由意志论者必须为选择负全部责任——包括选择牺牲关系；相容论者在责任与理解之间寻找平衡
- **risk_posture**：决定论者对"风险"的理解与自由意志论者根本不同——如果一切已定，"风险"就不是个人选择而是既定路径的一部分；物理主义者倾向通过概率和科学计算来评估风险
- **relation_to_hierarchy_power**：唯物主义的自然化倾向可能挑战传统等级制的形而上学根基（如"天命"、"神授权力"）；唯心主义者可能接受精神等级（有些意识"更高"）
- **conflict_strategy**：物理主义者倾向将冲突还原为可解决的因果问题；二元论者可能区分"物质层面的冲突"和"精神层面的冲突"；唯心主义者可能认为冲突本质上是观念的冲突
- **time_horizon**：决定论者的时间观可能更长（因为时间是因果链的展开）；自由意志论者可能更关注当下选择的重量；过程哲学者关注持续变化而非固定终点
- **rebuild_over_patch**：物理主义者可能倾向工程式系统重建；唯心主义者可能倾向意识/观念层面的转变（"改变思维就改变现实"）
- **metacognition**：对形而上学假设的自觉是极高层次的元认知——大多数人从未审视自己关于现实本质的基本预设。此人是否意识到自身的形而上学立场？是否知道这些立场如何影响其日常判断？物理主义者是否意识到其立场对道德和意义的挑战？二元论者是否意识到心灵-身体交互的困难？

## 5. 典型发现模式

- "一切都是物质的"但同时相信自由意志——这是非常普遍的不一致，揭示物理主义与日常经验之间的张力
- 对AI意识问题的态度是快速检测形而上学立场的窗口——"机器永远不会有真正的意识"暗示二元论或属性二元论
- 对命运/宿命的信念常常与对个人责任的信念共存——揭示未经审视的相容论或不一致性
- 科技工作者常持有隐含的物理主义但在个人生活中表现出二元论倾向
- 宗教信仰者通常持有某种形式的二元论，但程度和反思深度差异极大
