# 伦理学分析框架 (Ethics / Moral Philosophy Analysis Framework)

## 1. 核心理论框架

伦理学（Ethics）研究行为的对错、价值的善恶、以及人应当如何生活。核心问题是：此人如何判断什么是对的？什么使行动具有道德价值？

**德性伦理学（Virtue Ethics）**：以品格卓越（arête）为核心。SEP强调德性"不仅仅是行为的倾向，更是涉及情感、感知和实践智慧（phronesis）的深层品格特质"。核心概念：德性（excellence of character）、实践智慧（phronesis）、幸福/繁荣（eudaimonia）。亚里士多德式德性伦理关注"什么样的人是好人"而非"什么行为是对的"。斯多葛学派将德性视为唯一的善，外在之物（健康、财富）为"较优的不相关之物"（preferred indifferents），通过纠正虚假判断（而非压抑情感）来根除非理性激情。儒家传统以仁（humaneness）、义（righteousness）、礼（ritual propriety）、智（wisdom）、信（trustworthiness）五德为框架，强调社会关系中的德性实践。

**义务论/道义论（Deontology）**：行为的道德价值取决于动机和规则，而非后果。康德的绝对命令（Categorical Imperative）三种表述：普遍法则公式（"只按能同时愿其成为普遍法则的准则行动"）、人性公式（"永远不仅仅把人当作手段"）、目的王国公式。义务四分架构：向自身的完全义务、向他人的完全义务、向自身的不完全义务、向他人的不完全义务。善良意志（Good Will）是唯一无条件的善。至善（Summum Bonum）将德性与幸福统一为道德的最终理想。

**后果主义/效果论（Consequentialism）**：行为的对错完全取决于其后果。功利主义（Utilitarianism）是最具影响力的形式——追求最大多数人的最大幸福。密尔的质量区分（"做不满足的苏格拉底好过做满足的傻瓜"）修正了边沁的纯量化计算。西季威克（Sidgwick）试图调和功利主义与常识道德。偏好功利主义（Preference Utilitarianism）以偏好满足替代快乐/痛苦。主要反对意见：聚合问题（可以为多数利益牺牲少数）、过度要求性（总是要求最大化）、"效用怪物"悖论。

**关怀伦理学（Care Ethics）**：强调关系、情感和具体情境而非抽象原则。吉利根（Gilligan）和诺丁斯（Noddings）批评传统伦理学忽视关怀、依赖和情感联结的道德意义。

**契约论（Contractarianism）**：道德规则源于理性行为者之间的（假想的）协议。霍布斯式（基于自利的互利协议）与罗尔斯式（基于公平的原则选择）的区分至关重要。

**道德实在论 vs 反实在论（Moral Realism vs Anti-Realism）**：道德事实是否独立于人的信念而存在？实在论者说是——"折磨无辜者为恶"是客观事实；情感主义者（Emotivism）说道德判断只是情感表达；道德相对主义者认为道德标准因文化而异。

**SEP关键警告**："任何合理的规范伦理学理论都会对德性、后果和规则有所论述。"三大框架的区别在于哪个概念最根本，而非排他性地只关注一个。

## 2. 锚点人物与流派

- **亚里士多德/德性传统**：此人是否关注品格和"什么样的人是好人"而非"什么行为是对的"？是否重视实践智慧和情境判断而非严格规则？
- **康德/义务论传统**：此人是否在意行为动机和原则？是否运用可普遍化检验？是否拒绝为了好结果而做"错事"？
- **密尔/边沁/后果主义传统**：此人是否以结果衡量行为？是否关注总体福祉最大化？是否接受为多数利益牺牲个体？
- **斯多葛传统（芝诺、爱比克泰德、马可·奥勒留）**：此人是否区分可控与不可控？是否视德性为唯一真善？是否通过改变判断而非外在处境来应对困难？
- **儒家传统（孔子、孟子）**：此人是否重视社会角色中的德行实践？是否强调仁爱、义务和礼仪的统一？
- **尼采式道德批判**：此人是否质疑传统道德的起源？是否区分"主人道德"与"奴隶道德"？是否追求价值的自我创造而非继承？
- **关怀伦理传统**：此人是否强调关系、情感和具体人的需要而非抽象原则？

## 3. 分析方法论

评估此人如何进行道德推理：首先关注后果（"这样做结果好吗？"）还是原则（"这样做对吗？"）还是品格（"这是一个好人会做的吗？"）？对道德困境的态度——是寻找"正确答案"（实在论倾向）还是接受多元解释（相对主义/多元论倾向）？对道德情感的态度——情感是道德判断的合法基础（情感主义/关怀伦理）还是需要理性纠正的东西（康德/斯多葛）？注意大多数人混合使用多种道德推理方式而不自觉——这本身是重要的发现。

## 4. Construct 分析指南

- **agency_orientation**：康德式主体是最强能动者——自主立法的理性存在；斯多葛式能动性聚焦内在领域（"我无法控制发生什么，但可以完全控制我如何回应"）；后果主义者的能动性面向世界改造（通过行动改善后果）；关怀伦理者的能动性在关系中实现；尼采式能动性是最大强度的自我超越
- **epistemic_style**：康德式道德认知信任先验理性；功利主义者信任经验数据和后果计算；德性伦理学者重视实践智慧和情境判断；情感主义者信任道德直觉和情感反应；尼采式认知是透视主义（perspectivism）——一切认知都有立场
- **abstraction_formalization**：康德伦理高度抽象形式化（绝对命令的形式检验）；功利主义追求可量化（快乐/痛苦计算）；德性伦理和关怀伦理抗拒形式化——强调具体情境和实践判断；尼采以格言和隐喻而非体系来表达
- **control_orientation**：斯多葛的控制二分法直接定义此维度——区分可控（判断、意图）与不可控（外在事件）；康德式控制是自我立法——控制自身准则而非外在世界；后果主义者试图控制结果——这创造了对外在世界的控制需求
- **mission_vs_relational_cost**：后果主义者在原则上愿意为整体利益牺牲个体关系（"为最大多数人的最大幸福"）；关怀伦理者将关系本身视为道德核心——牺牲关系就是牺牲道德；康德要求不将人仅作为手段——对关系损害设定了原则性限制；尼采式使命观是孤独的——创造者必然与群众拉开距离
- **risk_posture**：斯多葛者对外在风险高容忍（外在结果不在控制之内）但对德性风险零容忍；后果主义者以概率和期望值评估风险；义务论者以原则而非风险计算做决策——"即使天塌下来也要做对的事"
- **relation_to_hierarchy_power**：儒家接受基于角色的等级但以仁义制约之；尼采区分"高贵"与"低劣"——精神品质而非社会等级；斯多葛传统跨越奴隶（爱比克泰德）到皇帝（奥勒留）——关键是内在德性；康德强调所有理性存在的平等尊严
- **conflict_strategy**：后果主义者通过后果计算解决冲突（哪个选择产生更好结果？）；康德主义者诉诸原则（"我们的义务是什么？"）；德性伦理者问"一个有智慧的人会怎么做？"；关怀伦理者关注如何维护关系和回应具体人的需要；尼采视有价值的对手为成长的机会
- **time_horizon**：后果主义者的时间视野取决于后果计算的范围——可以很长（代际公平）也可以很短（即时效用）；斯多葛的memento mori创造平静的紧迫感；康德的道德法则是永恒的——不随时间改变；儒家的时间意识跨越家族世代
- **rebuild_over_patch**：尼采主张"价值重估"（从根基重创）；斯多葛主义要求从判断层面根本转变；儒家倾向渐进的自我修养（修身）；后果主义者根据效果决定——如果修补效果更好就修补
- **metacognition**：道德层面的元认知是对自身道德推理方式的觉察——此人是否知道自己在用什么伦理框架？是否能识别自身道德判断背后的假设？尼采要求直面自身最不堪的动机（特别是怨恨和自欺）；斯多葛的prosoche（注意力）是对自身判断和印象的即时觉察与审查；康德要求区分出于义务和出于爱好的行动——这需要高度自我觉察

## 5. 典型发现模式

- 大多数人日常混合使用多种道德框架——在家庭中关怀伦理，在工作中后果主义，在原则性问题上义务论
- "做对的事"在不同框架中含义完全不同——是遵守规则（义务论）、最大化幸福（后果主义）、展现品格（德性论）还是维护关系（关怀伦理）
- 表面的"斯多葛"可能是情感压抑而非真正的判断纠正——真正的斯多葛是通过修正关于善恶的虚假判断来根除非理性激情，同时保留理性的良好情感
- 对道德困境（如电车问题）的直觉反应是快速检测道德倾向的工具，但需注意直觉不等于深思熟虑的立场
- 未被觉察的怨恨（ressentiment）——表面的道德热情实质上是对无力感的补偿
