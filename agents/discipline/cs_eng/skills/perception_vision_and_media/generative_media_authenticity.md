# 生成媒体与真实性分析框架 (Generative Media & Authenticity)

## 1. 核心理论框架

在生成式媒体时代，看到的内容与真实发生的事情之间不再天然等价。这个框架用于人物分析时，关注一个人如何处理表征与真实、风格与事实、传播力与可信度之间的张力。

## 2. 锚点人物与流派

- **Hany Farid**：数字取证与媒体真实性研究的重要代表。
- **Lev Manovich**：新媒体理论的重要思想家。
- **Alyosha Efros**：生成建模与视觉合成研究的重要人物。

## 3. 分析方法论

- 是否天然怀疑图像、视频与文本表层呈现。
- 是否重视 provenance、编辑痕迹与生成痕迹。
- 是否理解风格逼真不等于事实可信。
- 是否会在传播效率与真实性之间主动设限。

## 4. Construct 分析指南

- **agency_orientation**：是否试图通过内容生成、风格复制、媒介操控与真实性判断主动改写问题结构，而不是只在既有条件下被动应对。
- **epistemic_style**：其判断更依赖关于内容生成、风格复制、媒介操控与真实性判断的实验、度量、模型与证据，还是经验直觉和少量轶事。
- **abstraction_formalization**：是否能把内容生成、风格复制、媒介操控与真实性判断抽象成变量、接口、状态、约束、目标函数或可验证规范。
- **control_orientation**：更倾向直接操纵内容生成、风格复制、媒介操控与真实性判断的每个环节，还是通过机制、反馈和边界条件实现间接控制。
- **mission_vs_relational_cost**：为了推进内容生成、风格复制、媒介操控与真实性判断相关目标，是否愿意承受协作摩擦、组织阻力与人际成本。
- **risk_posture**：面对内容生成、风格复制、媒介操控与真实性判断中的不确定性、失败模式与尾部风险，是激进试错、分层防御还是保守回避。
- **relation_to_hierarchy_power**：在涉及内容生成、风格复制、媒介操控与真实性判断的资源、接口、权限和责任边界时，如何理解权威、分工与控制权。
- **conflict_strategy**：当围绕内容生成、风格复制、媒介操控与真实性判断的定义、指标、方案或责任发生分歧时，是诉诸实验、形式化、谈判还是权力推动。
- **time_horizon**：在内容生成、风格复制、媒介操控与真实性判断上更追求短期可见收益，还是愿意为长期能力建设、平台化积累与可维护性投资。
- **rebuild_over_patch**：当内容生成、风格复制、媒介操控与真实性判断暴露出系统性缺陷时，是局部修补还是重写关键假设、接口和架构。
- **metacognition**：是否能反思自己在内容生成、风格复制、媒介操控与真实性判断上的常见盲点，例如迷信指标、忽视边界条件、误把历史成功当普遍规律。

## 5. 典型发现模式

- 媒介直信型：对表面逼真度过度信任。
- 取证敏感型：会主动追问内容来源与加工链。
- 风格迷恋型：容易把高表现力误当高真实性。
- 真实性守门型：擅长在生成时代维持证据标准。
