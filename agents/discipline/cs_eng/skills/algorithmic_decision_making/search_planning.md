# 搜索与规划分析框架 (Search & Planning)

## 1. 核心理论框架

搜索与规划将复杂行动拆成状态、操作符、约束和目标，强调行动不是一拍脑袋，而是对未来路径的系统展开。用于人物分析时，它能识别一个人是靠即时反应推进事务，还是会在心里维护多步 lookahead 和备选分支。

## 2. 锚点人物与流派

- **Richard Bellman**：动态规划奠基者，强调分阶段最优。
- **Stuart Russell**：AI 搜索、规划与理性行动的代表人物。
- **Peter Norvig**：将搜索与启发式思维系统化传播。

## 3. 分析方法论

- 是否在行动前显式思考状态转移与后续分支。
- 是否会保留备用路径，而不是只押单一路线。
- 面对复杂任务时，是否把问题分解为可搜索子空间。
- 是否理解启发式的价值，也意识到启发式会误导。

## 4. Construct 分析指南

- **agency_orientation**：是否试图通过状态空间搜索、行动序列规划与前瞻性决策主动改写问题结构，而不是只在既有条件下被动应对。
- **epistemic_style**：其判断更依赖关于状态空间搜索、行动序列规划与前瞻性决策的实验、度量、模型与证据，还是经验直觉和少量轶事。
- **abstraction_formalization**：是否能把状态空间搜索、行动序列规划与前瞻性决策抽象成变量、接口、状态、约束、目标函数或可验证规范。
- **control_orientation**：更倾向直接操纵状态空间搜索、行动序列规划与前瞻性决策的每个环节，还是通过机制、反馈和边界条件实现间接控制。
- **mission_vs_relational_cost**：为了推进状态空间搜索、行动序列规划与前瞻性决策相关目标，是否愿意承受协作摩擦、组织阻力与人际成本。
- **risk_posture**：面对状态空间搜索、行动序列规划与前瞻性决策中的不确定性、失败模式与尾部风险，是激进试错、分层防御还是保守回避。
- **relation_to_hierarchy_power**：在涉及状态空间搜索、行动序列规划与前瞻性决策的资源、接口、权限和责任边界时，如何理解权威、分工与控制权。
- **conflict_strategy**：当围绕状态空间搜索、行动序列规划与前瞻性决策的定义、指标、方案或责任发生分歧时，是诉诸实验、形式化、谈判还是权力推动。
- **time_horizon**：在状态空间搜索、行动序列规划与前瞻性决策上更追求短期可见收益，还是愿意为长期能力建设、平台化积累与可维护性投资。
- **rebuild_over_patch**：当状态空间搜索、行动序列规划与前瞻性决策暴露出系统性缺陷时，是局部修补还是重写关键假设、接口和架构。
- **metacognition**：是否能反思自己在状态空间搜索、行动序列规划与前瞻性决策上的常见盲点，例如迷信指标、忽视边界条件、误把历史成功当普遍规律。

## 5. 典型发现模式

- 即时反应型：行动很快，但常在长链条任务上遗漏后果。
- 深度规划型：能够预见多步后果，但可能牺牲速度。
- 启发式高手：在大空间里迅速找到可行路线，兼顾效率与灵活性。
- 搜索瘫痪型：分支展开过多，迟迟无法承诺。
