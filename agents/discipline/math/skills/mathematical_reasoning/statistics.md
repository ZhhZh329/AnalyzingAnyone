# 统计学分析框架 (Statistics Framework)

## 1. 核心理论框架

统计学研究样本、总体、分布、估计、偏差、噪声与推断。用于人物分析时，它关注此人是否习惯用数量关系、比例变化、数据分布和代表性问题来理解现实，而不是被单个个案或情绪强度牵引。这个框架的核心落点是定量推理能力，即此人是否稳定地以数量级、趋势、样本意识和误差意识来支撑判断，并能区分“看起来重要”和“在分布中真正重要”之间的差别。

统计学视角强调两个关键纪律：区分信号与噪声，以及区分个例与总体。高统计倾向者不容易被单一成功故事、孤立失败案例或高情绪事件轻易带偏，他们更关心基线、样本偏差、分布尾部和估计误差；低此倾向者则更容易把个案当规律，或用模糊量词替代定量支撑。这个框架尤其适合分析一个人是否具备基率意识、样本意识和趋势判断能力。

## 2. 锚点人物与流派

- **Fisher**：强调实验设计与显著性推断，可用于观察此人是否在意证据质量和样本结构。
- **Neyman-Pearson**：重视检验、错误类型与决策阈值，可用于分析其是否有系统的证据判准。
- **Tukey**：强调探索性数据分析，可用于判断其是否通过数据模式而非预设立场理解问题。
- **Galton / modern statistical thinking**：提醒统计也可能带来过度简化，适合用于分析其量化倾向的边界。

## 3. 分析方法论

1. **识别定量表达习惯**：观察此人是否频繁引用数量级、比例、趋势、样本规模、基线比较和分布变化。

2. **检查样本与总体意识**：评估其是否能区分孤立个例与稳定趋势，是否有“样本是否代表总体”的意识。

3. **分析数据使用方式**：考察其面对数据时，是只摘取支持自己结论的数字，还是会讨论误差、偏差、异常值和局限。

4. **评估粗估与比较能力**：分析其是否能用粗估、区间和相对比较来支持现实判断，而不执着于伪精确。

## 4. Construct 分析指南

- **agency_orientation**：统计型能动性表现为能通过量化观察抓住系统变化杠杆，而非被叙事牵着走。
- **epistemic_style**：重点判断其是否信任样本、分布和趋势推断，而不只信直觉和单例。
- **abstraction_formalization**：看其是否能把模糊现象转成可计量变量和比较维度。
- **control_orientation**：统计视角下的控制更像“监测与度量”，通过指标看系统是否偏离。
- **mission_vs_relational_cost**：若高度量化，可能在沟通中弱化个体故事和关系感受。
- **risk_posture**：统计倾向强的人通常更敏感于基率与尾部风险，但也可能忽视低频高冲击的质性因素。
- **relation_to_hierarchy_power**：其可能更相信数据分布而不是职位高低，关注实际结果而非象征权威。
- **conflict_strategy**：冲突中可能诉诸样本、趋势和证据比较，而不是纯立场对撞。
- **time_horizon**：统计判断通常依赖时间序列和趋势积累，因此更容易形成中长期观察框架。
- **rebuild_over_patch**：当指标显示系统性偏差时，可能更倾向整体重调，而非只修补个别异常。
- **metacognition**：关键在于其是否意识到自己也会有选择偏差、幸存者偏差和叙事性误读。

## 5. 典型发现模式

- 强统计倾向者常说“这只是个例”“先看基线”“要看分布和样本”。
- 他们在决策中通常重视相对变化、数量级和趋势，但未必喜欢极端个案式叙事。
- 弱统计倾向者往往容易被高戏剧性案例或强烈印象带动判断。
- 过强量化倾向可能把“可测量的”错当成“最重要的”，从而忽视难以量化但关键的关系变量。
