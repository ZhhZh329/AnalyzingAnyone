# 客体关系分析框架 (Object Relations)

## 核心理论框架

客体关系理论关注个体如何把早期重要关系内化为稳定或分裂的“内在他人模型”。这些内在客体会持续影响亲密关系、权力感、分离焦虑与理想化/贬低模式。它比经典驱力理论更聚焦关系模板本身。

## 锚点人物与流派

- **Melanie Klein**：提出偏执-分裂位与抑郁位
- **Donald Winnicott**：足够好的母亲、过渡客体与真实/假自体
- **Otto Kernberg**：把客体关系与人格组织研究连接起来

## 分析方法论

- 看此人如何看待重要他人：整体、复杂，还是非黑即白
- 分析其理想化、贬低、黏附和愤怒循环
- 观察其是否能在失望中仍维持关系连续性
- 注意其对照料、依赖和分离的隐性情绪

## Construct 分析指南

- **agency_orientation**：稳定的内在客体有助于更稳的主体行动
- **epistemic_style**：是否能容纳他人的复杂性与模糊性
- **abstraction_formalization**：能否把重复关系脚本看作内部结构而非偶然事件
- **control_orientation**：分裂越强，越容易用控制对抗关系不确定
- **mission_vs_relational_cost**：内在客体不稳者常把关系波动带入使命执行
- **risk_posture**：分离与失望的耐受度会直接影响关系风险承担
- **relation_to_hierarchy_power**：权威者常被投射成全好或全坏客体
- **conflict_strategy**：一旦失望，是修复关系还是立刻翻转评价
- **time_horizon**：客体稳定性决定长期关系是否可持续
- **rebuild_over_patch**：深层关系模板改变常需要重整而非表层修补
- **metacognition**：能否看出“我对眼前这个人的反应里有旧关系的影子”

## 典型发现模式

- 理想化-贬低摆荡型：对人评价快速极化
- 假自体适应型：通过迎合关系环境维持稳定
- 客体恒常型：即便失望也能维持复杂而稳定的关系表征
- 分离脆弱型：关系距离变化会触发强烈不安
