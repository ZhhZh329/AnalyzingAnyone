# 条件作用学习分析框架 (Conditioning & Learning)

## 核心理论框架

条件作用框架关注行为如何在刺激、结果和联结中被塑造。它提供一种极其务实的视角：很多人格化叙述其实都可以还原为习得历史、强化路径与回避学习。人物分析中，这个框架特别适合解释稳定习惯与自动反应。

## 锚点人物与流派

- **Ivan Pavlov**：经典条件作用研究奠基者
- **B. F. Skinner**：操作性条件作用与行为塑造代表人物
- **John Watson**：行为主义早期代表人物

## 分析方法论

- 看此人的行为是否被固定线索自动触发
- 分析其回避、奖励和惩罚历史如何塑造当前模式
- 观察哪些行为被环境持续强化
- 区分信念解释与学习史解释

## Construct 分析指南

- **agency_orientation**：懂学习规律者更能主动重塑行为环境
- **epistemic_style**：更关注可观察行为与结果，而非抽象动机猜测
- **abstraction_formalization**：能否把行为拆成线索、反应、结果链条
- **control_orientation**：通过操纵强化条件来影响行为
- **mission_vs_relational_cost**：过度行为管理可能损伤关系体验
- **risk_posture**：惩罚历史会显著提升回避倾向
- **relation_to_hierarchy_power**：谁掌握强化资源，谁就拥有行为塑造权
- **conflict_strategy**：冲突也可被看作被强化的互动循环
- **time_horizon**：行为塑造依赖重复与延迟效果累积
- **rebuild_over_patch**：坏模式稳定后往往要重设强化环境
- **metacognition**：能否看见自己正在被什么线索和奖惩条件驱动

## 典型发现模式

- 奖励敏感型：对强化极快响应，容易形成稳定习惯
- 回避学习型：为了逃避不适而收缩行为版图
- 环境塑造型：善于通过改环境改行为
- 反复强化型：明知不合理但因为被奖励而继续
