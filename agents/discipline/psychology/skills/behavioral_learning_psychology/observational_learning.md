# 观察学习分析框架 (Observational Learning)

## 核心理论框架

观察学习强调人并不只通过直接奖惩学习，还会通过模仿、替代强化和榜样内化形成新行为。它适合分析人物如何被师长、同侪、偶像或组织文化悄悄塑形。

## 锚点人物与流派

- **Albert Bandura**：社会学习理论的核心人物
- **Walter Mischel**：延迟满足与社会学习研究的重要代表
- **Lev Vygotsky**：强调社会互动在学习中的中介作用

## 分析方法论

- 看此人主要模仿谁，以及为何模仿
- 分析其是否通过他人的成败迅速调整自身策略
- 观察其是否成为他人的行为模板
- 区分有意识学习与无意识同化

## Construct 分析指南

- **agency_orientation**：擅长观察学习者能更快借外部模型提升行动力
- **epistemic_style**：重视他人案例、榜样和替代经验
- **abstraction_formalization**：能否从榜样身上抽出可迁移机制
- **control_orientation**：会否通过挑选环境和样本来管理自我塑造
- **mission_vs_relational_cost**：模仿权威或榜样有时会压制真实自我
- **risk_posture**：通过替代经验可降低直接试错风险
- **relation_to_hierarchy_power**：谁是可见榜样，谁就拥有塑形权
- **conflict_strategy**：是否模仿到建设性或破坏性的冲突脚本
- **time_horizon**：长期暴露于某类榜样会深刻改变行为轨迹
- **rebuild_over_patch**：想改变自己时，常需要更换模仿对象和环境
- **metacognition**：能否指出自己正在学谁、复制什么

## 典型发现模式

- 榜样内化型：通过观察优秀模型快速升级
- 环境同化型：不知不觉学到周围群体的所有脚本
- 反面教材学习型：通过别人失败建立自己的边界
- 表层模仿型：只学姿态，不学底层机制
