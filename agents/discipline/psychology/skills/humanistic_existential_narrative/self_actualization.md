# 自我实现分析框架 (Self-Actualization)

## 核心理论框架

自我实现关注个体是否正在朝向自身潜能、价值和整合状态发展，而不是只满足外部成功标准。它并不等于精英主义的“成为最强”，而是指越来越完整、越来越一致、越来越不被外部脚本绑架。

## 锚点人物与流派

- **Abraham Maslow**：自我实现概念的核心提出者
- **Carl Rogers**：以机体评价过程和一致性深化该主题
- **Kurt Goldstein**：早期有机体自我实现思想的重要来源

## 分析方法论

- 看此人是在长成自己，还是越来越像环境要求的样子
- 分析其是否愿意发展被压抑的潜能和兴趣
- 观察其成功路径是否伴随更大完整感或更大异化
- 区分真正成长与高绩效表演

## Construct 分析指南

- **agency_orientation**：自我实现感强的人更像从内在方向出发行动
- **epistemic_style**：是否愿意相信自身深层偏好具有认识价值
- **abstraction_formalization**：能否描述自己想成为什么样的人
- **control_orientation**：自我实现不依赖过度控制，而依赖整合
- **mission_vs_relational_cost**：成为自己常要求承受误解和关系调整
- **risk_posture**：真正走向潜能发展往往要冒离开旧壳的风险
- **relation_to_hierarchy_power**：会否为了安全地位压抑自身成长方向
- **conflict_strategy**：冲突中是守住真实，还是退回假自我脚本
- **time_horizon**：自我实现是终身发展过程
- **rebuild_over_patch**：假自我太强时通常要做结构性重建
- **metacognition**：能否辨认哪些追求来自真实欲望，哪些来自他人期待

## 典型发现模式

- 潜能展开型：人生越走越有完整感
- 高绩效异化型：成就提升，但真实感下降
- 安全壳停留型：长期停在可预测但不成长的位置
- 真实化重建型：会主动拆除不再属于自己的身份脚手架
