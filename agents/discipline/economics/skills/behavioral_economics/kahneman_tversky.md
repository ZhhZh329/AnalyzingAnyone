# 行为经济学镜头：Kahneman & Tversky

## 核心理论框架

该镜头将人物理解为在不确定性下做选择的行动者，而不是稳定偏好被完整表达的理性机器。核心框架是 prospect theory、reference dependence、loss aversion 与 framing effects。分析重点不是此人“有没有某种心理偏差”，而是：当结果被表述为收益、损失、保全、回撤、机会成本或沉没成本时，此人的资源配置与风险承担是否系统性改变；此人的 choice pattern 是否受到 reference point 变化的显著影响；以及其口头偏好与实际选择是否持续错位。

对人物材料而言，这个镜头尤其适合处理 biography、interviews、public speech、documented decisions、organizational behavior 与 collaboration / conflict patterns 中反复出现的风险取向变化。重要的是观察“同一人面对不同 framing 时是否像在玩两套决策规则”，而不是机械列举 bias 名称。

## 锚点人物与流派

- **Daniel Kahneman**：强调决策不是围绕绝对财富水平，而是围绕 gains / losses relative to a reference point 展开
- **Amos Tversky**：强调 choice architecture、framing 与 judgment under uncertainty 如何改变表面一致的偏好表达
- **Prospect Theory tradition**：关注损失厌恶、概率加权、确定性效应，以及人在 gains domain 与 losses domain 中风险态度的可预测反转

## 分析方法论 / 证据中要看什么

1. **参考点如何形成**：看此人把什么当作 baseline。是现状、过往高点、公开承诺、组织目标、个人声望，还是某个 imagined future？如果 baseline 不稳定，很多表面“冲动”其实是 reference point shifting。
2. **收益域与损失域中的风险切换**：看其在上行阶段是否趋于保守、在下行阶段是否变得激进，尤其是连续受挫、 reputational damage、资源回撤或 deadline 压力之后的决策跃迁。
3. **framing sensitivity**：同一件事被描述为“保住已有成果”还是“争取额外增益”时，此人的选择是否发生系统变化。公开发言与内部行动不一致时，优先看 revealed preference。
4. **口头偏好与行动偏好差异**：有些人语言上强调长期主义、稳健或理性，但在 documented decisions 中 repeatedly choose downside-skewed gambles；也有人自称激进，却在关键节点高度规避损失。
5. **时间结构中的损失管理**：虽然该镜头不是纯粹的 intertemporal choice 理论，但可以观察此人是否把短期损失视为不可接受，还是愿意承受近期亏损以维护较高 reference point 或未来可选性。
6. **组织行为中的风险外溢**：看其是否将 loss-avoidance 压力转嫁给团队、合作方或下属，例如在 setback 后提高控制强度、改写考核口径或将风险集中到他人承担的边界上。

分析时优先使用 biography 中的转折事件、访谈里的决策自述、公开演讲里的 framing 语言、明确的组织决策、以及冲突中对损失与体面的反应。单句宣言的权重低于可重复观察到的 choice pattern。

## Construct 分析指南

- **risk_posture**：这是该镜头的核心构念。不要只问“敢不敢冒险”，而要问此人的风险承担是否依赖 gains vs losses 的 framing，是否在接近亏损、声望受损或 sunk-cost 高企时显著转向风险寻求。
- **time_horizon**：看此人是否为避免短期可见损失而牺牲长期选择空间，或反过来愿意承受当下亏损以保住未来高参考点。这里的重点是 intertemporal tradeoff 在 framing 下如何变化。
- **control_orientation**：部分强控制行为可能不是单纯的权力偏好，而是 loss containment。若此人一旦进入 perceived loss domain 就加码监督、集中审批或缩窄授权，这一构念可被该镜头有效分析。
- **metacognition**：只有当 repeated evidence 显示此人意识到自己的 reference points、framing vulnerability 与 losses-domain risk escalation，并会主动重构决策环境时，才适合上调。单次事后解释、单句反思或一次性的“我学到了教训”不够。
- **epistemic_style**：可做次级分析，重点看此人是否用不同 framing 讲述同一事实，以及其 belief updating 是否受可得损失的显著影响。
- **mission_vs_relational_cost**：若此人为了避免 mission loss 愿意接受显著关系损伤，可以提供中等强度解释，但仍应把焦点放在损失函数而非人格冷酷。
- **rebuild_over_patch**：只有当“推倒重来”明显出现在 losses domain 中，且被视为摆脱亏损参考点的手段时，这个构念才适合用该镜头解释。

## 共享构念强度地图

- **最强构念**：`risk_posture`、`time_horizon`、`control_orientation`
- **较弱但可分析的构念**：`metacognition`、`epistemic_style`、`mission_vs_relational_cost`、`rebuild_over_patch`
- **经常可能标记为 not_applicable 的构念**：`relation_to_hierarchy_power`、`abstraction_formalization`、`agency_orientation`

## 典型发现模式

- **损失域升级型**：在资源、地位或叙事控制受到威胁后，显著转向高风险决策，哪怕平时自述偏好稳健
- **参考点漂移型**：成功后迅速抬高 baseline，导致外界看来“永不满足”；本质上是 reference point 持续上移
- **语言偏好与行动偏好分裂型**：公开表述强调理性、长期、纪律，但在关键节点 repeatedly choose to avoid salient short-term loss
- **框架依赖型控制加码**：一旦问题被框定为“防止失去已有成果”，就加强审批、压缩容错、集中控制资源
- **亏损补偿型冒险**：在连续 setback 后采取非线性大动作，不一定因为高信心，而是因为 small repair 已无法恢复参考点

## 局限、常见误读与谨慎推断

- **局限**：该镜头擅长解释 observed choice under uncertainty，但不擅长单独解释制度结构、身份政治、长期组织合法性或纯技术能力差异。它回答的是“选择函数如何变形”，不是全部人格来源。
- **常见误读**：不要把任何高风险决策都叫作 loss aversion 的反面；也不要把任何保守决策都归因于 risk aversion。关键不在结果激进或保守，而在该行为是否相对于某个 reference point 发生系统偏移。
- **不要从稀疏证据过度推断**：单次豪赌、单次妥协、单句访谈、或媒体高度加工的事件，不能证明稳定的 framing dependence。至少需要跨情境材料显示 gains/losses framing 与 choice shift 之间存在重复关联。
- **不要误把修辞当作偏好**：有些公开发言只是 audience-facing framing，不等于真实决策规则。若口头说法与 documented decisions 冲突，优先信后者。
- **不要把经济选择模式心理化到过深层**：这个镜头可以讨论元认知和 choice architecture，但不应从有限证据直接推出深层人格创伤、病理性冲动或稳定临床特征。
